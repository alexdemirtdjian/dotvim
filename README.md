My vim configuration files :
============================

Install :
---------

	cd 
	git clone https://github.com/alexdemirtdjian/dotvim.git .vim

Making symlinks :
-----------------
	
	ln -s .vim/vimrc .vimrc
	ln -s .vim/gvimrc .gvimrc

Installing modules :
--------------------

	cd ~/.vim
	git submodule init
	git submodule update
